Why people pick DevFoundry

🔒

Local-first, actually

Your ideas, API keys, and every line of generated code stay on your laptop. No cloud builder, no telemetry, nothing to trust but your own machine.

🧠

Bring your own models

Anthropic, OpenAI, OpenRouter, Groq, Google — or point it at any OpenAI-compatible gateway, including self-hosted proxies.

📺

Nothing is a black box

Every stage streams its output live: documents, file diffs, test results, deploy logs. You can audit the factory while it works.

🖥

A real desktop app

8 MB native Tauri app. Manages its own Docker services — including starting Docker Desktop for you. No terminal required.

🧩

Isolated engines

Each pipeline stage runs in its own container, independently restartable and observable with one-click logs.

🎬

Demo mode built in

Try the full experience in 60 seconds with zero API keys and zero Docker — a scripted pipeline shows exactly what real runs look like.

Get started

# 1. Install — drag DevFoundry.app to Applications, or build from source:
git clone <repo-url> devfoundry && cd devfoundry/frontend
npm install && npm run tauri build

# 2. Try it instantly in demo mode (no Docker, no keys):
cd ../backend && python3 -m venv .venv && .venv/bin/pip install -r requirements.txt
DEVFOUNDRY_MOCK=1 .venv/bin/uvicorn app.main:app --port 9100

# 3. Open the app, type an idea, hit Forge It.
✓ spec → codebase → tasks → refine → deploy

Full pipeline needs Docker Desktop and one LLM API key — set both from the app's Settings page. Read the tutorial · architecture · roadmap.
