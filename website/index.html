Why DevFoundry

Cloud AI builders lock you in and take your code. DevFoundry keeps everything on your machine.

🔒

Local-first, actually

Your ideas, API keys, and every line of generated code stay on your laptop. No cloud, no telemetry.

🧠

Bring your own models

24 providers — Anthropic, OpenAI, Groq, Google, plus keyless OpenCode Zen and fully-local Ollama / LM Studio.

⚡

Zero-config

Nothing set up? It auto-detects a running local runtime and just works. Smart rotation & failover across providers.

Get started

# 1. Install — download the .dmg / .msi / .AppImage, or build from source:
git clone <repo-url> devfoundry && cd devfoundry/frontend
npm install && npm run tauri build

# 2. (Optional) a free local brain:
ollama pull qwen2.5-coder:14b     # DevFoundry auto-detects it

# 3. Open the app → pick a skill → type an idea → Forge It.
✓ spec → design → codebase → refine → deploy
